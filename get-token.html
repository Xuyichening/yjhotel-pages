🔑 云掌柜 Token 提取

提取登录凭证，用于看板实时数据对接

1
先在浏览器打开 v2.yunzhanggui.net，用你的账号登录云掌柜
2
登录成功后，把下面的蓝色按钮拖到书签栏（就是浏览器地址栏下面那一排）
3
在云掌柜页面上点击刚拖进去的书签，会弹出一个框，把内容复制发给小行

👇 按住拖到书签栏

🔑 提取Token
或者

登录云掌柜后，在地址栏输入以下内容然后回车：

javascript:void(prompt('',localStorage.getItem('userLogin')))

⚠️ 注意：需要在 javascript 前面手动敲字输入，直接粘贴可能被浏览器拦截
